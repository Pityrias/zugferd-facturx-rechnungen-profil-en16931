# Changelog

## 11.02.2026

**Erweiterung Version 0.2.1**

* Neue Funktion: Hinzufügen von zwei Pythonskripten zum einfachen Updaten der Übersetzungen
* Fehlerbehebung: Es wurde ein Fehler behoben, bei dem einige Übersetzungen nicht angezeigt wurden
* Fehlerbehebung: Es wurde ein Fehler behoben, bei dem ungültige Positionsdaten zu einem Abbruch des Makros geführt haben

**Extension Version 0.2.1**

* New Feature: Added two python scripts to simplify update of translation files
* Bugfix: Fixed a bug where some german translations were not being displayed
* Bugfix: Fixed a bug where invalid position data could cause the macro to crash

## 01.02.2026

**Erweiterung Version 0.2.0**

* Neue Funktion: Unterstützung für Überweisungsdaten (Felder BT-84, BT-85, BT-86 im Zugferd Standard 2.3.3)
* Fehlerbehebung: Es wurde ein Fehler behoben, bei dem der Block für Steuerbefreiung eingefügt wurde, obwohl keine Steuerbefreiung vorlag

**Extension Version 0.2.0**

* New Feature: Added support for credit transfer details (fields BT-84, BT-85, BT-86 in zugferd standard 2.3.3)
* Bugfix: Fixed bug where tax exemption section was inserted while no tax exemptions were present
